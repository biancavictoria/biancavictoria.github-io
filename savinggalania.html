What is Saving Galania?

Saving Galania is a visual novel where you play as Aria, a soldier, who goes back in time to save her teammates from a tragic future from a magic attack. You make choices to save her friends from their devastating destinies.

Roles: 2D Artist, Writer, Programmer

Team Size: 1 (for now!)

Project Length: 4 Weeks

Responsibilities: Create all 2D character art assets, Write the full narrative, Created the choices/paths in Tyranobuilder

Status: In Progress

Assets that I did not Create: Background assets, Music, UI

Summary of Responsibilities

I concepted and illustrated all 2D character art assets.

I wrote the four branching narratives and all character backstories.

I created the game using the visual novel engine Tyranobuilder.
